Detailed Description

Updates the grid by one timestep.

Function to update the densities of predator and prey in a grid instance according to the differential equation specified in the coursework pdf.

Parameters

rdouble instance representing the birth rate of hares (prey).
adouble instance representing the predation rate at which pumas (predators).
bdouble instance representing the birth rate of pumas (predators) per one hare (prey) eaten.
mdouble instance representing the puma (predator) mortality rate.
kdouble instance representing the diffusion rate for hares (prey).
ldouble instance representing the diffusion rate for pumas (predators).
deltaTdouble instance representing the timestep value.
